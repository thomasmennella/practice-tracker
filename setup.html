The Path
A quiet instrument for Theravāda practice

A few questions before you begin. Everything here stays on your device unless you choose to enable sync. Nothing is sent anywhere else, and there are no accounts.

Would you like to start at a specific meditation streak?
If you already keep a daily sitting practice, you can begin your streak at your true current count rather than at zero.
This is meant to reflect practice you've genuinely established — a starting honesty with yourself, not a number to inflate.
Would you like to start at a specific number of suttas read?
If you've already read part of the canon, you can seed that count so the tool reflects where you are.
As with the streak, this is a reflection of real reading, entered for your own sake.
Would you like to sync your practice across multiple devices?
Sync uses a free GitHub Gist as private storage, so your log can move between your phone and computer. It's optional — without it, everything simply stays on this device.
Setting up free Gist sync — one time, about 3 minutes
  1. Go to gist.github.com and sign in (or create a free GitHub account).
  2. Create a new gist: put anything in the description, name the file the_path_data.json, and type {} as the content. Choose "Create secret gist."
  3. Look at the URL of the gist you just made. It ends in a long string of letters and numbers — that's your Gist ID. Copy it and paste it below.
  4. Now create an access token: go to github.com/settings/tokens → "Generate new token" (fine-grained). Give it a name, set expiration as you like, and under Account permissions → Gists choose Read and write. Generate it and copy the token.
  5. Paste both below. They're stored only on this device.
You can also set this up later from Setup & Settings — choosing "No" now doesn't lock you out.
